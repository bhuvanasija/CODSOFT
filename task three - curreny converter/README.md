# 💱 Currency Converter - Java (Codsoft Internship)

A simple **Currency Converter** built in **Java** as part of the Codsoft Internship program.  
It supports conversion between multiple currencies using predefined exchange rates and provides a **Graphical User Interface (GUI)** built with **Java Swing**.

---

## 🚀 Features
- Convert between **INR, USD, EUR, GBP, JPY** (easily extendable).
- **GUI-based interface** for better usability.
- **Error handling** for invalid inputs.
- Exchange rates stored in a **HashMap** for quick lookup.
- Modular and beginner-friendly Java code.

---

## 🛠️ Tech Stack
- **Language:** Java (JDK 8+)
- **Libraries:** Swing (for GUI), AWT (for event handling)
